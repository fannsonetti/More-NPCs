# Legal Summary

This file is a plain-language summary of `LICENSE.md`. If there is any conflict, `LICENSE.md` controls.

## What People May Do

- Read the code.
- Download the repository.
- Build the mod for personal use.
- Make private local edits for personal use.
- Learn from the code and use it as reference.

## What People May Not Do Without Your Permission

- Reupload this repository anywhere else.
- Mirror the source code or binaries.
- Publish a fork containing this code.
- Repack or redistribute the source code.
- Upload edited versions of this project.
- Copy large, recognizable, or near-verbatim parts of this code, NPC content, generated data, or assets into another public project.

## Modpacks

Including the compiled mod in a modpack is allowed if you clearly credit the original author.

That does not allow:

- Reuploading the source code.
- Claiming the work as your own.
- Publishing modified versions of the project without permission.

## About "Drastic Changes"

Small edits do not make a copied release acceptable. Examples of changes that are still not enough on their own:

- Renaming files, classes, methods, variables, or NPCs.
- Reformatting code or moving code between files.
- Tweaking stats, schedules, dialogue, prices, or spawn locations.
- Adding a few features while keeping a substantial part of this project's code or content.

If someone wants to publish something based on this project, the safe rule is simple: ask permission first.

Permission requests may be sent to:

- `Fannsonetti` on Nexus Mods
- `fannso` on Discord

## When a Separate Public Project May Be Okay

A separately released project may be acceptable only if it is independently created and does not copy substantial protected material from this repository. Looking at the code to learn is allowed, but copying it 1:1 or closely enough that it is effectively the same is not. If there is doubt about whether something is too close, permission should be requested before release.

## Practical Enforcement Note

These files help you set clear terms and make enforcement easier, but no document can physically prevent copying by itself. What they do is make your rights and restrictions explicit so unauthorized uploads, forks, and reposts are much easier to challenge and remove.
