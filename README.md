# Fannso's MoreNPCs

`MoreNPCs` is a `Schedule I` mod that adds additional NPC content and related gameplay support.

This repository is source-available for reference and personal modding use only. It is **not** open source.

## Development Notes

- Target framework: `netstandard2.1`
- Build configurations: `Mono`, `Il2cpp`, `CrossCompat`
- Local `S1API` usage is controlled in `MoreNPCs.csproj`
- NPC documentation data is generated by `scripts/generate_npc_docs.py`
